## Overview
このアプリは税の計算を簡単にしてくれるアプリです。
今回はFlutter x Riverpod x Freezed x MVVM を用いてアプリを作成しました。
これまで使ったことがなかった技術だったので最初は苦労しましたがこのアプリ制作を通してRiverpodの使い方をより
理解できたと思います。

shareパッケージを使いシェアも可能にしました。

## Requirement
- macOS
- Flutter 2.8.0
- Dart 2.15.0

## Usage
基本的にtextfeildのhint text通りに数字を入力するだけ。
